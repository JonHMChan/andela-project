What is this?

This is where you get to see:

  • Tools you should use as a software developer
  • Information about a technology
All you have to do is click on the lists on the left

Each technology tag on the left comes with:

  • A summary about the language
  • IDE's suitable to use for that technology.
  • And Materials to read if you want to learn

TOOLS you should know about and use

  • Real Software developers use an IDE such as Sublime Text, etc
  • Real Software developers use a version control system such as GIT with Github or Bitbucket
  • Real Software developers use Google and Stackoverflow when they are stuck
  • Real Software developers make use of the browser developer console, for Mozilla Firefox, for Chrome, for Microsoft Edge or F12 for IE browsers
  • Real Software developers pair program with tools like Screenhero/Teamviewer for Mac and Windows, but Teamviewer alone for Linux, And collaborative tools such as CodeShare.io, Nitrous.io, Kobra.io and Coderpad.io
  • Real Software developers make use of the command line terminal extensively
  • Real Software developers know about Data Structures and Algorithms
  • And finally Real software developers would agree that when you go through either Bento.io or Free Code Camp You'd be considered a junior developer until you then have enough experience
